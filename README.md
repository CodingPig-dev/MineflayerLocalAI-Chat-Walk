# MineflayerLocalAI-Chat-Walk
MineflayerLocalAI-Chat&amp;Walk integrates Mineflayer with local models in GPT4All to create autonomous Minecraft bots. They can chat, walk, mine, and place blocks, simulating intelligent player-like behavior without cloud APIs, designed for experimentation and extensible AI-driven gameplay. And completly free to use!


This projekt includes official Mineflayer code in the lib folder.
